**Welcome to Quick.db!**

Quick.db is an open-source package meant to provide an easy way for beginners and people of all levels to access & store data in a low to medium volume environment. All data is stored persistently via either [better-sqlite3](https://github.com/WiseLibs/better-sqlite3) or [promise-mysql](https://www.npmjs.com/package/promise-mysql) and comes way various other quality-of-life features.

✅ **Persistent Storage** <br />
✅ **Multiple Drivers** <br />
✅ **Works out of the box** <br />
✅ **Beginner Friendly** <br />

## Installation

First, install the core package.

```md
npm install quick.db
```

You can choose either better-sqlite3 or promise-mysql as your driver. If you're not sure, you should use better-sqlite3.

### better-sqlite3

```js
const { QuickDB } = require("quick.db");
const db = new QuickDB();
```

### promise-mysql

```js
const { QuickDB, MySQLDriver } = require("quick.db");

(async () => {
    const mysql = new MySQLDriver({
        host: "localhost",
        user: "me",
        password: "secret",
        database: "my_db",
    });

    // Important: Connect to MySQL server before using it
    await mysql.connect();

    const db = new QuickDB({ driver: mysql });
})();
```
